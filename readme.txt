

To run the tests from Eclipse:
1) Install Java version 6 and Eclipse 
2) Unzip the files into a directory and import the existing project into Eclipse.
3) Copy apitests.properties to c:\ and change values to match your system
4) Run as unit test Alltests.java or to mimic command line run RestApi.java 

To run from command line:
1) Copy restTest.jar and forgeApiTest.cmd into a directory
2) Copy apitests.properties to c:\ and change values to match your system
3) Execute forgeApiTest.cmd
